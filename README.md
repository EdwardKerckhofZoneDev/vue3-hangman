# vue3-hangman

### Install

```bash
git clone https://github.com/EdwardKerckhofZoneDev/vue3-hangman.git
cd vue3-hangman
yarn install
yarn dev
```

### Demo

[https://hangman.zonedev.be/](https://hangman.zonedev.be/ 'Hangman demo')

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
